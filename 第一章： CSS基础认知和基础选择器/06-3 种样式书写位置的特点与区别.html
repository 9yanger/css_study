内嵌样式

书写位置：写在head结束标签前面的style标签中

作用范围：只针对当前页面

优先级：是由书写位置和选择器的优先级来决定

使用场景：小案例、对网站首次打开速度很高的网页（把首屏的css样式写在内嵌样式中）

问题：没有缓存

外链样式

书写位置：将 CSS 单独存为 .css 文件，然后使用 ``标签引入到页面中

作用范围：多个页面

优先级：是由书写位置和选择器的优先级来决定

使用场景：在实际开发项目中，外链样式是首先，公众样式的提取

问题：有缓存

行内样式

书写范围：直接放在html标签是style 属性中

优先级：优先级是三者中最高的

作用范围：配合JS完成特定元素样式的修改
